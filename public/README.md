# 🎯 Танчики Онлайн

Мультиплеерные "танчики" на Node.js + WebSocket. Один человек — хост
(запускает сервер), остальные подключаются по сети. Работает в браузере,
на компьютере и на телефоне.

## Что внутри

**Режимы:**
- ⚔ **Обычный бой (FFA)** — все против всех, до 12 игроков.
- 🛡 **Командный бой** — по коду комнаты, 2 команды (Красные/Синие) до 6
  человек: 4 нападающих + 2 саппорта. Раунд идёт **10 минут**, по окончании
  всем показывается таблица по киллам, карта перегенерируется и раунд
  начинается заново.

**Бой:**
- 100 HP, снаряд — 10 урона (10 попаданий на убийство).
- Снаряды быстрые, танки медленные. Задержка выстрела — 3 секунды.
- Патроны ограничены: 50 у нападающих/FFA, 35 у саппортов. Восполняются
  на возрождении.
- Возрождение — через 3 секунды, в случайной точке карты вне объектов.

**Только в командном бою:**
- 🩹 Точка лечения — лечит ~20 сек, КД точки 1:30.
- 🔵 Точка патронов — восполняет за 10 сек, максимум 5 раз за раунд, КД 1 мин.
- ⚡ **Бафф скорости x2 на 15 секунд** — спавнится в случайном месте карты
  каждые 30 секунд.

**Карта:** песчаная, огромная (3600×2400), генерируется заново каждый
раунд. Камни и ящики — препятствия, **кусты не мешают проезду и снарядам,
но прячут танк** — если стоишь в кустах, тебя видно только почти вплотную.

**Туман войны:** ники и снаряды других игроков видно только на среднем
расстоянии (в кустах — совсем близко). Свой танк и его снаряды видны всегда.

**Скины:** обычный (свой цвет), 🌈 радужный (переливается), 💡 неоновый
(светится, с подсветкой снарядов в тот же цвет). Выбираются на экране входа.

**Панель хоста** (иконка ⚙ внизу слева во время игры), пароль:
`jjjiiidddlllee` — кик и бан по IP, принудительный рестарт раунда.
Это открытый инструмент модерации, а не скрытый чит — все действия видно
по тому, что игрок вылетает из игры.

> Не стал делать секретное "мод-меню" с читами (ESP/заморозка врагов/
> замедление их пуль) — это тайное нечестное преимущество или троллинг
> против друзей же, с которыми играешь. Если нужна нечестная игра —
> это не ко мне, но кик/бан нарушителей и честные баффы — пожалуйста.

**Производительность:** карта кэшируется в один offscreen-canvas
(не перерисовывается объект за объектом каждый кадр), камера отдалена
(виднее карта), на мобильных пофикшен "прыгающий" интерфейс/скролл.

## Запуск

Нужен [Node.js](https://nodejs.org) 16+.

```bash
cd tanks-online
npm install
npm start
```

Появится:
```
Tanks Online сервер запущен!
Локально:     http://localhost:8080
Для друзей в одной сети: http://<твой-IP>:8080
Раунд: 600с | Лимит игроков: 12 | Пароль панели хоста: jjjiiidddlllee
```

Открой `http://localhost:8080` — попадёшь в игру. Друзья в той же
Wi-Fi заходят на `http://<IP-хоста>:8080`. Для игры через интернет нужен
проброс порта или туннель (ngrok / localtunnel) — напиши, помогу настроить
под конкретно твой хостинг.

## Настройки (server.js, начало файла)

| Параметр | Значение |
|---|---|
| Размер карты | 3600×2400 |
| HP / урон снаряда | 100 / 10 |
| Скорость танка / снаряда | 95 / 780 px/с |
| Задержка выстрела | 3000 мс |
| Патроны (FFA / нападающий / саппорт) | 50 / 50 / 35 |
| Длительность раунда | 600 сек (10 мин), тест: `MATCH_DURATION_SEC=30 node server.js` |
| Лимит игроков на комнату | 12 |
| Радиус видимости / в кустах | 650 / 160 px |
| Бафф скорости | x2, 15 сек, каждые 30 сек (только команда) |
| Пароль панели хоста | `jjjiiidddlllee` |
